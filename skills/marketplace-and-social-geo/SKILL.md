# Marketplace & social GEO

Many SMEs don't have a website they can edit — their storefront is **Amazon,
Alibaba, Etsy, eBay**, and their presence is **LinkedIn, YouTube, Instagram,
TikTok, 小红书/抖音**. AI increasingly pulls from exactly these. You can't add
schema to Amazon, but you *can* make these profiles the clear, consistent,
quotable source AI cites. (This is about profiles **you control** on platforms;
for earned third-party mentions like Reddit/Wikipedia, see `earn-mentions`.)

Read `shared/references/geo-principles.md` and `shared/references/ai-engines.md`
first.

## The levers you DO control on platforms

### Marketplace listings (Amazon / Alibaba / Etsy / eBay / 1688)
- **Title + first lines = the answer.** Put what it is, who it's for, and the key
  spec in plain words up front — AI lifts these.
- **Complete every field**: specs, materials, dimensions, use cases, compliance —
  structured attributes are machine-readable and quotable.
- **Q&A and reviews**: seed real Q&A; earn honest reviews (huge trust signal AI
  uses for products).
- **Consistent brand name** across all listings so AI links them to one entity.

### LinkedIn company + people
- Complete company page: clear one-line description, what you do/for whom,
  website, specialties. AI cites LinkedIn for "what is <company>".
- Founder/team profiles consistent with the brand (entity + trust).

### YouTube
- Real demo/explainer/factory-tour videos; titles matching questions buyers ask;
  accurate descriptions + links. Increasingly surfaced and cited.

### Social profiles (Instagram / TikTok / 小红书 / 抖音 / Facebook)
- Complete bio (what/who/where/link); consistent name/handle; pinned content that
  states the offer clearly. AI and AI-search pull from social for brand and
  product understanding.

## Make them work together (entity consistency)

The big win: **same brand name, same description, same key facts, same links
everywhere.** Inconsistency makes AI unsure which entity is real and splits your
authority. List every profile, align NAP/brand/description, cross-link them, and
point them all at one canonical hub (even a simple link-in-bio or one landing
page).

## How to run it

1. **Inventory** every platform/social presence you have.
2. **Audit** each: is the offer in plain text up front? fields complete?
   consistent brand/description/links?
3. **Fix** titles/bios/fields to be answer-first and consistent; earn reviews;
   add a clear Q&A.
4. **Cross-link** + point to one hub.
5. Spot-check: ask AI "what is <brand>" / "best <product> for <use>" and see which
   profile (if any) it cites.

## Output

- A presence inventory + per-profile fix list (answer-first text, completeness,
  consistency, reviews).
- The biggest inconsistency to fix first.
- A simple "one hub" plan if they have no real website.

## Bilingual / 出海 (中文)

国内 SME 的门面常是 **1688 / 淘宝/天猫 / 抖音 / 小红书 / 视频号 / 公众号**;出海是
**Amazon / 阿里国际站 / 独立站 / LinkedIn / TikTok**。打法一样:标题与简介用文字
讲清"做什么/给谁/规格/价格",字段填满,口碑真实,品牌名与描述各平台一致。出海
B2B 供应商另见 `export-supplier-geo`。

## Refuse / push back when

- Asked to keyword-stuff titles, fake reviews, or run inconsistent fake brands →
  refuse; platforms penalize it and it confuses the entity you're building.
